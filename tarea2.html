Ejercicio 2

Búsqueda de un PPA y revisión de paquetes

En este ejercicio se trabajó con el PPA oficial de LibreOffice, que distribuye versiones actualizadas para sistemas basados en Ubuntu, incluyendo Linux Mint.

1. Acceso al PPA en Launchpad

Se accedió al PPA mediante navegador para consultar su contenido:

https://launchpad.net/~libreoffice/+archive/ubuntu/ppa

Captura 1: Visualización del PPA en Launchpad.

2. Añadir el PPA desde terminal

A continuación, se añadió el repositorio al sistema mediante el comando:

sudo add-apt-repository ppa:libreoffice/ppa

Captura 2: Incorporación del PPA a Linux Mint.

3. Actualizar la caché del sistema

Para que el sistema reconozca los nuevos paquetes, se ejecutó:

sudo apt update

Captura 3: Descarga de información desde el PPA.

4. Consultar los paquetes ofrecidos

Finalmente, se realizó una búsqueda de los paquetes disponibles mediante:

apt search libreoffice

Captura 4: Listado de paquetes del PPA.

Conclusión

Se ha localizado un PPA válido, se ha añadido al sistema y se ha verificado que contiene múltiples paquetes relacionados con LibreOffice, cumpliendo así con los objetivos del ejercicio.
