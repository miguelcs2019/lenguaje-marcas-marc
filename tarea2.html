Ejercicio 1

Activación de un repositorio externo y comprobación del paquete

El objetivo de este ejercicio consistió en activar un repositorio externo en el sistema, actualizar la caché de APT y comprobar que el paquete proporcionado por dicho origen estaba disponible para su instalación.

Dado que en las versiones actuales de Linux Mint el repositorio Partner y el antiguo plugin de Flash han quedado obsoletos, se utilizó un repositorio equivalente para cumplir el objetivo técnico del ejercicio: el repositorio oficial de Google Chrome, el cual proporciona el paquete google-chrome-stable.

1 y 2. Añadir la clave GPG y el repositorio

wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo apt-key add -
echo "deb [arch=amd64] http://dl.google.com/linux/chrome/deb/ stable main" | sudo tee /etc/apt/sources.list.d/google-chrome.list

Captura: Adición de la clave GPG del repositorio y registro del fichero en sources.list.d.

3. Actualización de la caché

Una vez añadido el repositorio se actualizó la caché del gestor de paquetes:

sudo apt update

4. Comprobación del paquete proporcionado

Se verificó la existencia del paquete mediante la herramienta apt:

apt search google-chrome

Captura: Actualización de APT y detección del paquete google-chrome-stable.

5. Comando de instalación del paquete

sudo apt install google-chrome-stable

Conclusión

Se añadió un repositorio externo, se actualizó el catálogo y se verificó que el paquete proporcionado estaba disponible, cumpliendo los requisitos del ejercicio.

Ejercicio 2

Búsqueda de un PPA y revisión de paquetes

En este ejercicio se trabajó con el PPA oficial de LibreOffice, que distribuye versiones actualizadas para sistemas basados en Ubuntu, incluyendo Linux Mint.

1. Acceso al PPA en Launchpad

https://launchpad.net/~libreoffice/+archive/ubuntu/ppa

Captura: Visualización del PPA en Launchpad.

2. Añadir el PPA desde terminal

sudo add-apt-repository pppa:libreoffice/ppa

Captura: Incorporación del PPA a Linux Mint.

3. Actualizar la caché del sistema

sudo apt update

Captura: Descarga de información desde el PPA.

4. Consultar los paquetes ofrecidos

apt search libreoffice

Captura: Listado de paquetes del PPA.

Conclusión

Se ha localizado un PPA válido, se ha añadido al sistema y se ha verificado que contiene múltiples paquetes relacionados con LibreOffice.

Ejercicio 3

Instalación del repositorio de WoeUSB y localización del fichero .list

WoeUSB es una herramienta utilizada para crear memorias USB booteables con imágenes ISO de Windows. En este ejercicio se debía añadir el repositorio correspondiente y localizar el fichero que lo describe dentro del sistema.

1. Añadir el PPA al sistema

sudo add-apt-repository ppa:tomtomtom/woeusb

Captura: Añadiendo el PPA de WoeUSB desde la terminal.

2. Actualizar la caché del sistema

sudo apt update

3. Localizar el fichero .list asociado al PPA

Los ficheros que describen los repositorios se almacenan en:

/etc/apt/sources.list.d/

Para mostrar el fichero asociado al repositorio se utilizó:

cat /etc/apt/sources.list.d/tomtomtom-ubuntu-woeusb-*.list

Captura: Visualización del archivo en sources.list.d.

Conclusión

Se identificó correctamente la ruta del fichero encargado de describir el repositorio añadido.

Ejercicio 4

Diferencias entre apt upgrade y apt full-upgrade

Ambos comandos mantienen el sistema actualizado aunque presentan diferencias a nivel de dependencias y gestión de paquetes instalados.

apt upgrade

  • Actualiza paquetes instalados.
  • No elimina paquetes existentes.
  • No instala paquetes adicionales si son requeridos.
  • Comportamiento conservador.

apt full-upgrade

  • Actualiza el sistema como apt upgrade pero puede instalar dependencias nuevas.
  • Puede eliminar paquetes para completar la actualización.
  • Permite la instalación de kernels y transiciones de metapaquetes.
  • Comportamiento agresivo y resolutivo.

Conclusión

apt upgrade mantiene el sistema actualizado sin eliminar paquetes, mientras que apt full-upgrade puede modificar dependencias para completar actualizaciones más profundas.
