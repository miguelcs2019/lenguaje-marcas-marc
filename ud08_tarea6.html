Ejercicio 3 – Sistema de archivos ReFS

ReFS (Resilient File System) es un sistema de archivos diseñado por Microsoft para entornos de servidor, enfocado en la integridad y fiabilidad de los datos.

Su uso como destino de copias de seguridad es recomendable debido a su capacidad para detectar errores, su escalabilidad y su resistencia frente a la corrupción de datos.

Actividad de ampliación – Cliente NFS en Windows

El comando Install-WindowsFeature NFS-Client permite instalar en Windows Server la funcionalidad de cliente NFS, haciendo posible el acceso a carpetas compartidas desde servidores Linux.

Una vez instalado el cliente NFS, es posible montar una carpeta remota como una unidad de red (por ejemplo, la unidad L:), facilitando la interoperabilidad entre sistemas Windows y Linux.

Esta funcionalidad resulta especialmente útil en entornos empresariales mixtos, donde conviven servidores con distintos sistemas operativos.
